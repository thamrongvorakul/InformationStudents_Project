Automated Nuget

file script upload to chocolatey.

API

For upload file to chocolatey we provide api with few steps.

STEP1  create folder for file upload.
First you must create folder for file upload by post method.

POST : http://etappkgmgr.emea1.ciscloud:81/postcreatefolder

Example
AngularJS

var nameOfFolder = { value : "Examplename"}

$http.post('http://etappkgmgr.emea1.ciscloud:81/postcreatefolder', nameOfFolder)
.success(function(data, status, headers, config)){
// do something.
}
.error(function(data, status, headers, config)){
// do something.
}

STEP2  post file upload.
Second you must should upload file by post method.

POST : http://etappkgmgr.emea1.ciscloud:81/postupload

Example
AngularJS

Upload.upload({
url: 'http://etappkgmgr.emea1.ciscloud:81/postupload',
file : file
}).progress(function (evt)){
// do something.
})
.success(function(data, status, headers, config){
// do something.
})

PS. See also : AngularJS file upload https://github.com/danialfarid/ng-file-upload

STEP3  post feed.
Third you must send feedname and command script by post method.

You can separate command script with semi-colon ;

POST : http://etappkgmgr.emea1.ciscloud:81/postfeed

Example
AngularJS

var feedname = {
feedname : "etap"
cmd : "Write-Host hello, world"
}

$http.post('http://etappkgmgr.emea1.ciscloud:81/postfeed', feedname)
.success(function(data, status, headers, config)){
// do something.
}
.error(function(data, status, headers, config)){
// do something.
}

Fourth you must send all of metadata by post method.

POST : http://etappkgmgr.emea1.ciscloud:81/postdata

Upload execution file to Chocolatey

You can upload any execution files to chocolatey, type of file that uploaded is .nupkg and you can download the uploaded file via http://etappkgmgr.emea1.ciscloud:81/nuget/
